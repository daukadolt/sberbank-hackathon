Сбербанк API

# Назавание методов URL Краткое объяснение
1 GET htttp://localhost:9200/bank_accounts/ Получить список всех банковских аккаунтов.
2 GET htttp://localhost:9200/bankaccounts/:id Получить уникальный ID указанного банковского аккаунта
3 POST htttp://localhost:9200/bankaccount Создание нового банковского аккаунта
4 PUT htttp://localhost:9200/bankaccounts/:id Обновить детали банковского аккаунта.
5 DELETE htttp://localhost:9200/bankaccounts/:id Удалить банковский аккаунт с ID указанным в URL.
6 GET htttp://localhost:9200/bankaccounts/:id/balance Получить баланс банковского аккаунта с ID указанным в URL.
HTTP статус Значение
200OK Запрос получен и успешно обработан
202Accepted Запрос был получен и будет обработан в ближайшее время
400Bad Request Что-то не в порядке с запросом, проверьте правильность запроса, чтобы устранить ошибку
401Unauthorized У вас нет доступа к запрошенным данным
403Forbidden Ошибка аутентификации, время действия токена истекло или запрос находится за пределами доступа
404Not Found Запрошенный ресурс не существует
500Server Error Произошла ошибка со стороны сервера. Пожалуйста перезагрузите страницу позднее.
